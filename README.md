# Introduction
This project was designed based on Algorithms & Data Structure method by JavaScript language. 
Minesweeper is a game which requires the player to open all cells on the board in order that you won’t touch unrevealed bombs under cells. 
This game will clearly show the DSA course properties, the use of array data structure and algorithms to solve problems. 
In more detail, use recursion to check neighboring squares and open them.

# Basic rules
When the player starts Minesweeper, choose your desired level, the screen will show a table of small squares that are considered as minefield.
Choose a square on the board, if you hit a bomb, you lose. If that cell does not contain bomb, there will be an area of cells that will be opened with numbers.
When the player thinks a tile contains bomb, place a flag on that square.
If there are already enough mines corresponding to the number on that center square, all the remaining squares have no mines.
The player will win when he opens all the cells without mines.

# Player guidance
Finding your first mine by left click at any random place
Right click to flag a spot and remove flag
Left clicking on numbers
The game is implemented with 3 levels: easy, medium, hard
